📊 Sales Performance Dashboard (Excel)

📌 Project Overview
This project presents an interactive Sales Performance Dashboard built using Microsoft Excel.  
It helps analyze sales data across different regions and track overall performance using clear and meaningful visualizations.

🔹 Key Features
- Region-wise filtering using slicers  
- Identification of top 5 sales performers  
- Target Achievement % vs Away From Target % analysis  
- Daily sales tracking (Day 1 to Day 5)  
- Interactive charts (Bar, Pie, Line)  

🛠 Tools & Techniques Used
- Microsoft Excel  
- Formulas (Total Sales, Target %, Calculations)  
- Slicers (for dynamic filtering)  
- Charts & Data Visualization  

📈 Business Insights
- Quickly identify high and low performers  
- Analyze sales trends across regions  
- Track target achievement effectively  
- Support better decision-making  


📂 Files Included
- Sales Dashboard Excel File (.xlsx)  
- Dashboard Screenshot  

## 🚀 Outcome
This dashboard converts raw sales data into interactive and easy-to-understand insights, enabling faster and smarter business decisions.
